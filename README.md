# 🐢 マイペース

時間割・カレンダー・週間ToDo・毎日の習慣トラッキングで「焦らず一つずつ確実に」進むための、個人用スケジュール管理PWA。

## 使い方

1. ブラウザで公開URLを開く
2. iPhone: Safariの共有ボタン →「ホーム画面に追加」/ Android: Chromeのメニュー →「アプリをインストール」
3. 毎日の通知は、iOSショートカットのオートメーション（毎日指定時刻 → URLを開く）で設定する

## データについて

時間割・予定・ToDoなどのデータはすべて**端末内（localStorage）にのみ保存**され、外部には一切送信されません。このリポジトリに含まれるのはアプリのコードだけです。

## 構成

- `index.html` — アプリ本体（自己完結型）
- `manifest.webmanifest` / `sw.js` / `icon-*.png` — PWA用ファイル（ホーム画面追加・オフライン対応）
